# ☁️ Cloud Native Resource Monitoring Python App on Kubernetes (K8s)

A fully guided project that takes you from **local Python development** to **production-grade deployment** on **AWS + Kubernetes**.  
This repository demonstrates how to build, containerize, store, and deploy a lightweight **System Monitoring App** using **Flask**, **Docker**, **AWS ECR**, and **Amazon EKS**.

---

## 🚀 Key Features

- 🧪 Python Monitoring App using `Flask` + `psutil`
- 📦 Dockerized Application (production-ready Dockerfile)
- 🏷️ AWS ECR Integration using Terraform
- ☸️ Kubernetes Deployments & Services
- ☁️ EKS Cluster Provisioning (AWS Managed Kubernetes)
---

## 🎯 Project Workflow

| Stage | Description & Key Outcomes |
|------|-----------------------------|
| **1. Application Development** | Create a minimal system monitoring app using Flask and collect metrics using `psutil`. |
| **2. Containerization** | Package the Python app into a Docker image using a simple Dockerfile. |
| **3. AWS Image Management** | Use AWS ECR to create repositories and push Docker images securely. |
| **4. Cloud Native Infrastructure** | Provision an Amazon EKS cluster with node groups for Kubernetes workloads. |
| **5. Deployment on Kubernetes** | Deploy app using Deployment + Service manifest (NodePort or LoadBalancer). |
